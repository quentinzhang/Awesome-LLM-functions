## 目标

OpenAI的函数调用机制（Function calling）十分强大，但函数的开发和调试都比较麻烦，互联网也缺少这方面的工具和资源。

本项目旨在创建一个“开箱即用”的共享函数集，并提供一些函数开发相关的资源和技术框架，让开发者可以更容易地将外部函数与大语言模型的API集成起来，实现自己想要的功能。
